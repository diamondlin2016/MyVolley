# MyVolley

练手项目。



### 实现的需求
- 支持请求 JSON 文本类型学，音频，图片类型，批量下载。上传~
- 请求各种数据时，调用层不用关心上传参数的封装
- 获取数据后，调用层不用关心 JSON 数据的解析
- 回调时，调用层只需要知道传入的 JSON 的对应响应类
- 回调响应结果发生在主线程（线程切换）
- 对下载，上传扩展
- 支持高并发请求，请求队列一次获取，可以设置最大并发数，设置先请求先执行

### 会用到的知识点
- 泛型
- 请求队列
- 阻塞队列

### 用到的设计模式

- 模板方法模式
- 单例模式
- 策略模式
- 生产者消费者模式
